# Case Study Reference Paper

This folder contains the published paper used as the worked case study:

To QG, Stanton R, Schoeppe S, Doering T, Vandelanotte C. *Differences in
physical activity between weekdays and weekend days among U.S. children and
adults: Cross-sectional analysis of NHANES 2011-2014 data.* Preventive Medicine
Reports. 2022;28:101892. https://doi.org/10.1016/j.pmedr.2022.101892

The included PDF metadata identifies the article as open access under the
Creative Commons Attribution 4.0 International licence:

https://creativecommons.org/licenses/by/4.0/

The paper is included so readers can trace how the decision-tree case-study
answers, dataset reconstruction rules, and visualisation examples were derived.
